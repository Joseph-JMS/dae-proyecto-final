Bienvenido a la Aplicación de Gestión de Estudiantes

Esta aplicación permite administrar información de estudiantes de manera rápida y sencilla. Podrás realizar operaciones fundamentales como registrar nuevos estudiantes, editar datos existentes, eliminarlos o consultar la lista completa en cualquier momento.

Usa el menú superior para navegar entre los módulos principales y comenzar a gestionar la información de manera eficiente.

Registrar Estudiantes

Agrega nuevos estudiantes al sistema con toda su información básica.

Editar y Actualizar

Actualiza datos como nombre, edad, curso y demás información relevante.

Eliminar Registros

Administra el sistema eliminando estudiantes que ya no deben figurar.

Listado Completo

Consulta todos los estudiantes registrados y navega entre sus detalles.
